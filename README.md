# find_duplicates

## How to Run?

### Pre-Requisites:

- You should have python 3.7+ installed

### Steps:
- Inside the project directory run the following command
```
python -m venv venv
```

- Then run,
```
 pip install -r requirements.txt
```

- Then run,

For windows -
```
venv\Scripts\activate
```

For Linux/Mac -
```
source venv/bin/activate
```

- Finally
```
python main.py
```
